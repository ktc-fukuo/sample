関連情報

主キー・外部キーで、関連するモデルを管理します。

親子モデル
親モデルの主キーに、採番キーを一つ加え、子モデルの主キーとする。
兄弟モデル
兄弟モデル間で、同一の主キーを持つ。
履歴モデル
親子モデルのうち、変更理由カラムの有無に関わらず、主キー以外の項目が一致する。
変遷モデル名が履歴モデル名に前方一致する。
参照モデル
参照評価用の「キーサフィックス：値サフィックス」に該当するモデルを、外部キーで参照している。
自エンティティの主キーを外部キーとしている場合は再帰する。
複合モデル
２以上のテーブルの主キーのうち、一致個数が最大となる主キーを組み合わせて、複合モデルの主キーとする

    ■複合元モデル          ■複合モデル
    ┌─────┐          ┏━━━━━┓
    │複合元１ID│─────┃複合元１ID┃
    │----------│    ┌──┃複合元２ID┃
    │複合元１名│    │┌─┃複合元連番┃
    └─────┘    ││  ┃----------┃
                      ││  ┃複合先名  ┃
    ┌─────┐    ││  ┗━━━━━┛
    │複合元２ID│×  ││
    │----------│    ││
    │複合元２名│    ││
    └─────┘    ││
                      ││
    ┌─────┐    ││
    │複合元２ID│──┘│
    │複合元連番│───┘
    │----------│
    │複合元２名│
    └─────┘
転生モデル
外部キーが必須項目であり、他のテーブルには存在しない

■転生元モデル          ■転生先モデル
┌─────┐          ┏━━━━━┓
│転生元ID  │──┐    ┃転生先ID  ┃
│----------│    │    ┃----------┃
│転生元名  │    │    ┃転生先名  ┃
└─────┘    └──┃転生元ID  ┃
                        ┗━━━━━┛
派生モデル
外部キーが必須項目であり、他のテーブルにも存在する

■派生元モデル          ■派生先モデル
┌─────┐          ┏━━━━━┓
│派生元ID  │──┐    ┃派生先１ID┃
│----------│    │    ┃----------┃
│派生元名  │    │    ┃派生先１名┃
└─────┘    ├──┃派生元ID  ┃
                  │    ┗━━━━━┛
                  │    ┏━━━━━┓
                  │    ┃派生先２ID┃
                  │    ┃----------┃
                  │    ┃派生先２名┃
                  └──┃派生元ID  ┃
                        ┗━━━━━┛
集約モデル
外部キーが必須項目でなく、集約先となるモデルが１つしかない。

■集約先モデル          ■集約元モデル
┏━━━━━┓          ┌─────┐
┃集約先ID  ┃──┐    │集約元ID  │
┃----------┃    │    │----------│
┃集約先名  ┃    │    │集約元名  │
┗━━━━━┛    ├──│集約先ID  │
                  │    └─────┘
              ×─┘
